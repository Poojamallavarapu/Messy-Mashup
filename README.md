#  Messy Mashup – Music Genre Classification

This project focuses on building a robust deep learning system for **music genre classification under noisy mashup conditions**.
- Kaggle Link : https://www.kaggle.com/competitions/jan-2026-dl-gen-ai-project

##  About the Project
- Developed for the **Messy Mashup competition**
- Classifies audio into **10 different music genres**
- Handles **noisy and mixed audio inputs**
- Works on mashups created from multiple stems such as vocals, drums, bass, and other sounds

##  Models Used
- **CNN from Scratch**
- **Pretrained Audio Spectrogram Transformer (AST)**
- **Pretrained EfficientNet-B0**

##  Key Features
- Mel Spectrogram based audio processing
- Synthetic mashup generation
- Noise augmentation using ESC-50
- Transfer learning with pretrained models
- Test-Time Augmentation (TTA)

##  Tech Stack
- Python
- PyTorch
- Torchaudio
- Transformers
- torchvision
- Weights & Biases

Built as part of **Generative AI course project work**.
